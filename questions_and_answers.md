###### 1. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let greeting;
greetign = {};
console.log(greetign);
```

- A: `{}`
- B: `ReferenceError: greetign is not defined`
- C: `undefined`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: A

<i>"let greeting;" initializes the greeting variable with the value undefined. "greetign = {};" attempts to assign an empty object to a variable named "greetign", as "greetign" variable doesn't exist it creates a new global variable "greetign" and assigns an empty object to it.</i>

</p>
</details>

###### 2. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sum(a, b) {
  return a + b;
}

sum(1, "2");
```

- A: `NaN`
- B: `TypeError`
- C: `"12"`
- D: `3`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C

<i>The output of the sum function will be 12, and it type will be string. It happens because of JavaScript automatic type conversion. When JavaScript tries to operate on a "wrong" data type, it will try to convert the value to a "right" type. Type of number 1 will be converted to string from number.</i>

</p>
</details>

###### 3. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
const food = ["🍕", "🍫", "🥑", "🍔"];
const info = { favoriteFood: food[0] };

info.favoriteFood = "🍝";

console.log(food);
```

- A: `['🍕', '🍫', '🥑', '🍔']`
- B: `['🍝', '🍫', '🥑', '🍔']`
- C: `['🍝', '🍕', '🍫', '🥑', '🍔']`
- D: `ReferenceError`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: A

<i>In this context, the 'info' object's 'favoriteFood' property is initialized with the first element of the 'food' array. Assigning a new value to 'info.favoriteFood' does not affect the 'food' array because it does not create any reference or connection to the 'food' array. Instead, it only changes the value of the 'info' object's property.</i>

</p>
</details>

###### 4. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sayHi(name) {
  return `Hi there, ${name}`;
}

console.log(sayHi());
```

- A: `Hi there,`
- B: `Hi there, undefined`
- C: `Hi there, null`
- D: `ReferenceError`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: B

<i>If a function is called with missing arguments, the missing values are set to "undefined". Here, "sayHi()" is called without any arguments. As a result, the function returns the string "Hi there, undefined", where "undefined" is the value of the "name" parameter inside the function.</i>

</p>
</details>

###### 5. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let count = 0;
const nums = [0, 1, 2, 3];

nums.forEach((num) => {
  if (num) count += 1;
});

console.log(count);
```

- A: 1
- B: 2
- C: 3
- D: 4

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C

<i>Here, in this "forEach" loop if the value of num is true then count will increment by 1. We know "0" is a falsy value, hence for the value of num = 0 count value didn't increment. And for the other values of the array "nums" count value incremented.</i>

</p>
</details>
